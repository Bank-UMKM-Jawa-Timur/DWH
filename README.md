## System Requirements

-   PHP version 8.0 - 8.2

## Installations

1. Clone repository.
2. Run command "composer install".
3. Run command "php artisan migrate --seed".
4. Run command "php artisan serve" to run the project.
5. Open [http://127.0.0.1:8000](http://127.0.0.1:8000) to open the project.
